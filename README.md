# MDK Core

MDK is a JavaScript-based backend SDK that provides a modular, extensible foundation for monitoring mining infrastructure, controlling devices and containers, collecting telemetry and operational data, building custom mining applications and integrations.

---

⚠️ **Work in Progress**

MDK is currently under active development and is **not yet considered stable**.

At the moment, the **latest development code is available in the `develop` branch**, while the `main` branch will eventually contain stable releases once development reaches a stable milestone.

---

## Current Development Status

- `develop` → Active development branch (latest features and changes)
- `main` → Reserved for stable releases

If you want to test the SDK or explore the latest functionality, please use the **`develop` branch**.

---

## Getting the Latest Development Code

Clone the repository and switch to the `develop` branch:

```bash
git clone https://github.com/tetherto/mdk-be.git
cd mdk-be
git checkout develop
```

## Contributing

Contributions are welcome while MDK is in development.

To contribute:

1. Fork this repository
2. Clone your fork locally
3. Create a new branch from `develop`
4. Make your changes
5. Submit a Pull Request targeting the `develop` branch

Example:

```bash
git checkout develop
git checkout -b feature/your-feature-name
```
Once your changes are ready, open a Pull Request against the **develop** branch for review.

## Reporting Issues

If you encounter bugs, unexpected behavior, or have feature requests, please open an **Issue** in this repository with:

- A clear description of the problem
- Steps to reproduce (if applicable)
- Relevant logs or screenshots

## Roadmap

MDK is currently evolving and the API surface may change while development is in progress. Once MDK stabilizes:

- The `main` branch will contain stable tagged releases
- Documentation will be expanded
- Versioned releases will be published
